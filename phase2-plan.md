# Phase 2 작업 문서 — 클러스터 인과 + 데이터 무결성

> 목적: Phase 1 스크리닝(100사)을 **10~20사**로 압축.
> 방법: **게이트 → Quality×Value 점수 → 클러스터 인과(CC) 축 복원**.
> (필드워크는 Phase 3로 유보. 이 문서는 전부 데스크 리서치 범위)

## 0. 후보군 확정 로직 (요약)

- **시작 풀**: 상장 100사 (Phase 1 판정 라벨은 *필터가 아니라 주석*으로만 사용 → 요검증도 재평가)
- **레드라인 게이트**: 상장폐지 / 자기자본<40% / OPM<8% / 클러스터無 / 전공정(스코프밖) → **생존 47사**
- **점수**: **Quality × Value (QARP)**, 균형 50:50
  - Quality = 0.5·재무(F) + 0.5·기술·모트(T)
  - Value = 0.6·(시총/영업익 백분위) + 0.4·(PSR 백분위) ← 쌀수록 고득점
- **민감도 코어 7** (가중치 흔들어도 top12 견고): 일본전자재료·미쓰이금속·테라프로브·우에무라·시바우라메카·어드밴테스트·야마이치전기
- **스코프밖 각주**: 레이저텍(전공정, 종합 최상급이나 스코프밖)·교세라(OPM 컷)

## 2. 클러스터 인과(CC) 프레임워크

### 2.1 왜 후공정이 지방에 집적하는가 (이론 = 리포트의 척추)

전공정(前工程)은 자본집약·중앙집중형이라 소수 메가팹에 모이고, 소재도 전 세계 배송이 가능하다.
반면 **후공정·소재·장치는 "팹 근접성 자체가 경쟁우위"** 다:

1. **고빈도 소모품 납품** — 다이싱 테이프·슬러리·본딩와이어 등은 JIT·근접 물류 우위.
2. **장치 현장서비스** — 설치·유지보수·수율 튜닝에 팹 인근 거점이 필수.
3. **OSAT·조립검사** — 팹 산출물을 받아 처리 → 물리적 근접이 리드타임·수율 협업에 직결.
4. **선단 패키징 공동개발** — FC-BGA·칩렛은 팹-소재-장치사의 co-engineering 밀도가 높아 대면 근접이 유리.

→ 후공정 생태계는 앵커 팹 주변에 **집적할 구조적 유인이 전공정보다 크다.**
이것이 "후공정 니치 × 지방 클러스터"가 **우연이 아니라 인과**인 이유.

### 2.2 앵커 매핑

| 클러스터 | 앵커 | 성격 | 생존사 수 |
|---|---|---|--:|
| 규슈 | TSMC 구마모토(JASM, 2024 가동) | **현재형 카탈리스트** | 36 |
| 홋카이도 | 라피더스 치토세(2nm, 2025~) | **미래형 카탈리스트** | 5 |
| 도호쿠 | 기존 반도체·전자 집적 | **기반형** | 6 |

### 2.3 CC 점수식 (전부 공개 정보)

**CC = 0.40·입지정밀도 + 0.35·앵커결합 + 0.25·시계열증거**

- **입지정밀도**: 본사+주력공장 클러스터 내 =1.0 / 주력공장만 =0.8 / 영업·서비스거점만 =0.4 / 고객만 클러스터 =0.1
- **앵커결합**: 앵커 팹 직접 공급 확인 =1.0 / 앵커 인근 동종 팹 공급 =0.6 / 간접·추정 =0.3 / 불명 =0
- **시계열증거**: 앵커 투자 *이후* 증설·진출·수주 공시 확인 =1.0 / 실적 반응 정황 =0.5 / 없음 =0

**검증 출처**: 有価証券報告書「事業所の状況」(본사·공장 위치) / 기업 IR·설비투자 릴리스 / 앵커 관련 수주 뉴스.

### 2.4 왜 이게 두 문제를 동시에 푸는가

- **변별력 복원**: 현재 규슈 36사가 뭉쳐 있어 클러스터가 점수에서 죽어 있음(그래서 QARP에선 게이트로만 씀). 입지정밀도·앵커결합을 넣으면 *"도쿄 본사+규슈 영업소 1개"* 와 *"주력공장이 JASM 인근+구마모토 증설 발표"* 가 갈라짐 → 종합 점수에 CC를 **제3축으로 복귀** 가능.
- **인과 입증**: 라벨이 아니라 시계열 논증이 됨 → 심사의 *"왜 하필 지방인가"* 에 답.

## 4. 다음 실행 순서

1. **CC 데이터 채우기** — 상위 ~25사부터 有報·IR로 본사/주력공장/앵커/시계열 4열 채움 → CC 점수 산출.
2. **QARP + CC 재합성** — 최종 종합 = f(Quality, Value, CC). 가중치는 채운 뒤 민감도 재분석.
3. **Tier A 무결성** — 간토전화 성장률 등 후보군 3건 확정(CC 채우며 병행).
4. **Tier B 반영 여부 결정** — 6249→6265 정정 / 4960 제외를 Phase 1 데이터에 반영할지 사용자 확인.
5. 최종 10~20사 확정 → 페이지/데이터 구현.
