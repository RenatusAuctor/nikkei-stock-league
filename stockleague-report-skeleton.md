## 1. 투자 테마와 사회적 의의 — ◐ 재료 있음, 사회성 축 전면 배치 필요
- 테마: **"후공정 니치 톱 × 지방 클러스터"** — 세계 1등급 기술이 도쿄가 아닌 구마모토·아오모리·후쿠시마에 있다.
- **사회성(심사 배점 축 — 기존 문서에선 암묵적이었던 것을 명시)**:
  - **지방창생**: 앵커 팹 유치 → 지방 고용·투자·생태계(히라타=구마모토 토박이 기업의 증산, 닛토방적=후쿠시마 100명+ 고용 보도, 마이크로닉스=아오모리 1981년부터).
  - **경제안보**: 반도체 공급망의 일본 내 재구축(경산성 공급확보계획 인정 — 닛토방적), TSMC·라피더스 국책 유치.
  - "投資로 사회 이슈를 읽는다"는 대회 취지와 정합 — 우리 포트폴리오 자체가 지방창생·경제안보의 진행 지표.
- 소스: [phase2-report.md](phase2-report.md) §1 / index 허브 서사.

## 2. 학습 과정 (우리가 어떻게 배웠나) — ☐ 집필 필요(재료는 풍부)
심사가 좋아하는 "과정의 정직한 기록" — 이미 문서에 다 있다:
- 100사 스크리닝 → 게이트 → 47사 → 앵커 검증 → 14사 (퍼널 숫자로).
- **실패와 교훈을 숨기지 않기**: ① 시총 자동 재조회 15배 오독 사고와 폐기(plan §6.5) → "표시값 인용+정합 체크+스크립트 검증"으로 개선 ② 간토전화를 1년 묵은 기사로 오판했다가 최신 결산으로 복귀(plan §5) ③ 클러스터 태그의 대량 반증(plan §3.5 — 태그≠검증).
- 750조합 민감도(plan §6.4): "기준 숫자를 바꿔도 결론이 버티는가"를 스스로 공격해 봄.

- **비중 산식과 논거**: qarpNow 비례 + 상한 10%/하한 4% + 수수료 1.1% 차감([phase3-runbook.md](phase3-runbook.md) §4). "비싸진 종목은 빼는 게 아니라 가볍게 든다."

- **장기보유 이해**: 클러스터 형성 = 수년 단위 구조 변화 → 가설이 장기보유를 요구. 수수료 구조도 회전에 불리. 전 종목 배당 실시(평균 이율 ~1.2% — 보조 논거). 리셋 불사용 원칙(runbook §7).
