## 0. 후보군 확정 로직 (요약)

- **시작 풀**: 상장 100사 (Phase 1 판정 라벨은 *필터가 아니라 주석*으로만 사용 → 요검증도 재평가)
- **레드라인 게이트**: 상장폐지·비상장 / 자기자본<40% / OPM<8% / 클러스터無 / 전공정(스코프밖) → **생존 47사**
  ※ 클러스터 게이트는 **Phase 1 가설 태그 기준의 스크린**(가설 정합 필터)이다. 실제 입지·공급 링크는 §3.5~3.8에서 개별 검증했고, 검증된 인과만 `anchor`(확정 앵커)로 인정한다 — 태그≠검증을 혼동하지 않는다(예: 미쓰이금속은 태그 규슈·실입지 사이타마 → 앵커 아님).
- **점수**: **Quality × Value (QARP)**, 균형 50:50
  - Quality = 0.5·재무(F) + 0.5·기술·모트(T)
  - Value = 0.6·(시총/영업익 백분위) + 0.4·(PSR 백분위) ← 쌀수록 고득점
- **민감도 코어 7** (가중치 흔들어도 top12 견고): 일본전자재료·미쓰이금속·테라프로브·우에무라·시바우라메카·어드밴테스트·야마이치전기
- **스코프밖 각주**: 레이저텍(전공정, 종합 최상급이나 스코프밖)·교세라(OPM 컷)

## 2. 클러스터 인과(CC) 프레임워크

### 2.1 왜 후공정이 지방에 집적하는가 (이론 = 리포트의 척추)

전공정(前工程)은 자본집약·중앙집중형이라 소수 메가팹에 모이고, 소재도 전 세계 배송이 가능하다.
반면 **후공정·소재·장치는 "팹 근접성 자체가 경쟁우위"** 다:

1. **고빈도 소모품 납품** — 다이싱 테이프·슬러리·본딩와이어 등은 JIT·근접 물류 우위.
2. **장치 현장서비스** — 설치·유지보수·수율 튜닝에 팹 인근 거점이 필수.
3. **OSAT·조립검사** — 팹 산출물을 받아 처리 → 물리적 근접이 리드타임·수율 협업에 직결.
4. **선단 패키징 공동개발** — FC-BGA·칩렛은 팹-소재-장치사의 co-engineering 밀도가 높아 대면 근접이 유리.

→ 후공정 생태계는 앵커 팹 주변에 **집적할 구조적 유인이 전공정보다 크다.**
이것이 "후공정 니치 × 지방 클러스터"가 **우연이 아니라 인과**인 이유.

### 2.2 앵커 매핑

| 클러스터 | 앵커 | 성격 | 생존사 수 |
|---|---|---|--:|
| 규슈 | TSMC 구마모토(JASM, 2024 가동) | **현재형 카탈리스트** | 36 |
| 홋카이도 | 라피더스 치토세(2nm, 2025~) | **미래형 카탈리스트** | 5 |
| 도호쿠 | 기존 반도체·전자 집적 | **기반형** | 6 |

### 2.3 CC 점수식 (전부 공개 정보)

**CC = 0.40·입지정밀도 + 0.35·앵커결합 + 0.25·시계열증거**

- **입지정밀도**: 본사+주력공장 클러스터 내 =1.0 / 주력공장만 =0.8 / 영업·서비스거점만 =0.4 / 고객만 클러스터 =0.1
- **앵커결합**: 앵커 팹 직접 공급 확인 =1.0 / 앵커 인근 동종 팹 공급 =0.6 / 간접·추정 =0.3 / 불명 =0
- **시계열증거**: 앵커 투자 *이후* 증설·진출·수주 공시 확인 =1.0 / 실적 반응 정황 =0.5 / 없음 =0

**검증 출처**: 有価証券報告書「事業所の状況」(본사·공장 위치) / 기업 IR·설비투자 릴리스 / 앵커 관련 수주 뉴스.

### 2.4 왜 이게 두 문제를 동시에 푸는가

- **변별력 복원**: 현재 규슈 36사가 뭉쳐 있어 클러스터가 점수에서 죽어 있음(그래서 QARP에선 게이트로만 씀). 입지정밀도·앵커결합을 넣으면 *"도쿄 본사+규슈 영업소 1개"* 와 *"주력공장이 JASM 인근+구마모토 증설 발표"* 가 갈라짐 → 종합 점수에 CC를 **제3축으로 복귀** 가능.
- **인과 입증**: 라벨이 아니라 시계열 논증이 됨 → 심사의 *"왜 하필 지방인가"* 에 답.

## 6. 선정 기준의 논리적 근거 (2026-07-03) — "왜 이 기준이어야만 하는가"

사용자 질문("각 기준·수치의 근거가 탄탄해야 하는 것 아닌가")에 대한 답. **기준의 종류(범주)**와 **정확한 수치**는 근거의 성격이 다르므로 분리해서 다룬다.

### 6.1 범주별 논리적 필연성 (게이트가 이런 종류여야 하는 이유)

| 게이트 | 막는 실패 | 이게 없으면 생기는 문제 |
|---|---|---|
| 상장폐지·비상장 제외 | 매수 불가능한 종목 혼입 | 포트폴리오에 "살 수 없는 종목"이 들어감 — 숫자 문제 아니라 이진 자격 요건 |
| 자기자본<40% 제외 | 재무 취약 종목 | 경기 하강기 생존 불확실 종목이 "니치 1위"라는 이유만으로 포함됨 |
| OPM<8% 제외 | 무수익 니치 리더 | "세계 점유율은 높으나 돈을 못 버는" 종목이 포함 — 투자 가설(경쟁우위→수익)과 모순 |
| 클러스터 태그無 제외 | 가설 무관 종목 | "후공정×지방클러스터"를 검증할 수 없는 종목이 섞여 논증이 흐려짐 |
| 전공정 제외 | 스코프 이탈 | 레이저텍 등 최상급 전공정주가 들어가면 "후공정 니치" 테마 규율이 깨짐 |

→ 이 5가지는 **"이런 종류의 위험/모순을 막아야 한다"는 논리로 필연적**이다. 어떤 게이트를 하나라도 빼면 구체적으로 위 표의 문제가 재발한다.

### 6.2 수치 자체의 근거 — "정확히 그 숫자"는 증명 불가, 대신 실측했다

연속값 임계값(40%, 8%)에 "이 숫자가 유일하게 옳다"는 제1원리 증명은 존재하지 않는다(Piotroski F-score, Fama-French 팩터 분할 등 표준 퀀트 방법론도 관행값·표본 백분위를 쓴다). 대신 두 가지를 실측했다:

**(a) 표본 내 위치(모집단 97사 기준)**
- 자기자본 40% = **하위 9퍼센타일** → 명백한 재무부실만 제거하는 **관대한 컷**
- OPM 8% = **하위 38퍼센타일** → 표본의 1/3 이상을 제거하는 **선별적 컷**, 투자 가설의 핵심(수익성)이라 엄격하게 설정한 것과 정합

**(b) 임계값 민감도 그리드 (자기자본{30,35,40,45,50%} × OPM{5,6,7,8,10,12%} = 30개 조합)**
- **핵심(core) 8사** — 30개 조합 전부에서 백본15에 생존: 일본전자재료·미쓰이금속·우에무라·시바우라메카·어드밴테스트·야마이치전기·다이요HD·후소화학. → 이 8사는 "40%/8%라서"가 아니라 **재무·경쟁력이 압도적이라 임계값과 무관하게 견고**.
- **조건부(conditional) 7사** — 정확한 컷라인에 따라 포함/제외가 갈림: 테라프로브·간토전화·일본가이시·에스펙·오르가노·일본특수도업·일본필러. → 이 부분은 "40%/8%를 채택했기 때문에 포함"이라고 정직하게 등급을 낮춰 표시(`tier:"conditional"`).
- 앵커 오버레이 4사(호리바·TOK·디스코·로체)는 수치 게이트와 무관 — 확정 인과 링크로 별도 선정.

### 6.3 결론 및 산출물 반영
"왜 이 숫자여야만 하는가"에는 정직하게 답할 수 없지만(그런 절대 근거는 없음), **"이 숫자를 흔들어도 결론이 얼마나 안 바뀌는가"는 실측 가능**하고 그게 방금 한 일이다. `phase2-data.js`에 `tier`(core/conditional/anchor) 필드로 반영, 대시보드에 핵심/조건부 구분 필터·표시·해설 박스 추가 완료(2026-07-03).
