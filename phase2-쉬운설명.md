# Phase 2 팀 브리핑 — 쉽게 읽는 버전

> 팀원들이 **가장 먼저** 읽는 문서. 어려운 용어 없이 우리가 뭘 했는지 설명함.
> (자세한 근거·숫자는 [phase2-plan.md](phase2-plan.md), 리포트용 서사는 [phase2-report.md](phase2-report.md)에 있음. 이 문서부터 읽고 그 다음에 봐도 됨.)

---

## 0. 우리가 하는 게 뭔데?

닛케이 스톡 리그 = **"이런 주제로 이런 회사들에 투자하겠다"는 리포트를 쓰는 대회.**
중요한 건 돈을 얼마 벌었냐가 **아니라**, 주제가 얼마나 좋고 우리가 얼마나 잘 조사했냐임.

그래서 우리 승부처는 **① 좋은 주제 + ② 직접 발로 뛴 조사**임.

---

## 1. 우리 팀의 투자 아이디어 (한 문장)

> **"일본 시골에 거대한 반도체 공장이 새로 생기는데, 그 공장 옆에 붙어 있는 세계 1등 부품·장비 회사들이 이 붐을 타고 큰다."**

이걸 사면 돈이 될 거다 — 이게 우리 주장임.

---

## 2. 왜 이 아이디어가 말이 되나 (핵심 논리)

반도체 만드는 과정은 크게 둘로 나뉨:

- **앞공정(前工程)**: 웨이퍼(반도체 원판)를 만드는 단계. 돈 엄청 들고, 큰 공장 몇 개에 몰려 있음.
- **뒷공정(後工程)**: 그 원판을 **자르고 → 포장하고 → 검사하는** 단계. 그리고 여기에 쓰는 **소재·장비**.

우리는 **뒷공정**에 집중함. 왜냐면 뒷공정 회사들은 **공장 바로 옆에 있어야 유리**하기 때문:

1. 소모품(테이프·약품 등)을 **자주** 납품해야 함 → 가까워야 함
2. 장비 **고장 나면 바로 달려가** 고쳐야 함 → 가까워야 함
3. 공장이 만든 걸 **받아서** 처리함 → 가까워야 함
4. 새 기술을 공장이랑 **같이 개발**함 → 얼굴 보고 해야 유리

→ 그래서 뒷공정 회사들은 큰 공장 주변에 **모여드는 게 당연**함. 이건 우연이 아니라 **이유가 있는 현상**이라는 게 우리 논리의 핵심.

우리가 노리는 큰 공장(= "앵커", 사람 끌어모으는 자석 같은 존재):
- **규슈 구마모토** → TSMC(대만 반도체 1등 기업) 공장, 2024년 가동 시작
- **도호쿠 이와테** → 키옥시아(일본 메모리 기업) 공장

## 3. 100개 회사 → 14개로 어떻게 줄였나 (3단계)

우리는 처음에 후보 회사 **100개**를 갖고 있었음. 여기서 3번 걸러서 **14개**로 압축함.

### 1단계 — 기본 탈락 (100개 → 47개)
아래 중 하나라도 걸리면 바로 컷:
- 주식을 **살 수 없음** (상장폐지 등)
- **돈을 못 버는** 회사 (영업이익률 8% 미만)
- **재무가 약한** 회사 (자기자본 40% 미만)
- 우리 주제랑 **상관없는 업종** (앞공정 등)

### 2단계 — "진짜로 공장 옆에 있나?" 확인 (47개 → 14개)
여기가 제일 중요하고, 우리가 제일 고생한 부분.

47개 회사가 다들 "우리 규슈 근처예요" 라고 이름표(태그)는 달고 있었는데,
**공식 서류로 하나씩 확인해보니 실제로는 도쿄·오사카에 있는 곳이 태반**이었음.
세계 1등이어도 공장 옆에 없으면 우리 이야기랑 안 맞으니까 **가차없이 뺐음.**
→ 진짜로 공장 옆에 있는 게 확인된 **14개만** 남김.

### 3단계 — 순위 매기기
그 14개를 **"좋은 회사인데 주가도 안 비싼 순"**으로 줄 세움. (이 점수를 우리끼리 'QARP'라고 부름)

**"신규"** 표시 4개는 이번에 새로 찾아낸 회사. ~~재무 숫자 재확인이 아직 안 끝남~~ → **확인 끝(2026-07-05)**: 4개 전부 최신 결산 숫자와 일치했음. 참고로 히라타기공은 이익률 8.77%로 탈락 기준(8%)을 **턱걸이**로 넘었다는 것도 정직하게 적어둠 — 얘는 숫자보다 "본사가 구마모토"라는 위치 증거로 뽑힌 회사임.

**추가로 정직하게 적어두는 것 2개:**
- **주가가 그새 올랐음.** 점수 매길 때 쓴 시가총액은 예전 값이라, 2026-07-03 기준 최신 몸값을 다시 다 확인했고, **47개 전체를 지금 가격으로 다시 채점까지 끝냈음**(대시보드 「지금 종합」 열, 2026-07-05). 좋은 소식: **1등(일본전자재료)은 지금 기준으로도 1등**이고 테라프로브 3등·스미토모BL 5등·히라타 9등 — 우리 핵심은 여전히 안 비쌈. 나쁜 소식: 시바우라메카·닛토방적은 그새 많이 올라서 "싸다" 매력이 사라짐(40위권으로 하락). 그래도 14개 선정은 "공장 옆" 기준이라 안 바뀜. 종목별로 돈을 얼마씩 담을지는 **이 지금 점수 기준**으로 Phase 3에서 정함. (참고: 주가가 *떨어져서* 점수가 좋아진 회사도 있는데 — 싸진 데는 이유가 있을 수 있어서 그 점수만 보고 좋다고 판단하지 않음.)
- **14개가 전부 "순수 뒷공정"은 아님.** 8개는 뒷공정 직결, 5개는 앞·뒷공정 공통 인프라(초순수·배관 같은 것), 1개(도쿄오카)는 앞공정 주력+패키징 소재 병행. "앞공정 전용 장치" 회사만 주제 밖으로 뺐고, 이 기준을 규칙으로 문서에 못 박음(→ "왜 얘는 되고 쟤는 안 돼?"에 답 가능).

## 5. 일부러 뺀 회사들 — 이게 우리 무기임

세계 1등인데도 **공장 옆에 없어서 뺀** 회사가 9개 있음
(미쓰이금속·우에무라공업·어드밴테스트·야마이치전기·간토전화·일본가이시·에스펙·후소화학·일본특수도업).

이 중엔 점수(QARP) 순위가 **최상위인 것도 있는데 뺐음.**

왜 이게 무기냐면 → 심사위원이 *"너네 그냥 좋아 보이는 회사 마음대로 고른 거 아냐?"* 라고 물을 때,
**"아뇨, 좋은 회사여도 주제에 안 맞으면 뺐어요"** 라고 답할 수 있음.
= 우리가 **원칙대로 골랐다**는 증거. (이걸 리포트에 "정직한 반증"이라고 씀.)

## 7. 어려운 용어 사전 (상세 문서 볼 때 참고)

| 문서에 나오는 말 | 쉬운 뜻 |
|---|---|
| 후공정 / 뒷공정 | 반도체를 **자르고·포장하고·검사**하는 단계 |
| 앵커 | 주변에 회사들을 끌어모으는 **큰 공장** (TSMC 등) |
| 클러스터 | 회사들이 **한 지역에 모여 있는 것** |
| 게이트 | **탈락 관문** (못 넘으면 컷) |
| 레드라인 게이트 | 1단계 기본 탈락 관문 |
| 확정 앵커 게이트 | 2단계 "진짜 공장 옆인지" 확인 관문 |
| QARP | **"좋은 회사 × 싼 주가"** 점수 |
| 밸류(Value) | 주가가 **싼지 비싼지** |
| 퀄리티(Quality) | 회사가 **튼튼하고 기술 좋은지** |
| 750조합 민감도 | 기준 숫자를 바꿔봐도 **같은 회사가 살아남는지** 확인한 것 (= 대충 안 골랐다는 증명) |
| verified:false | **재무 재확인 아직 안 됨** 표시 (지금은 14개 전부 확인 완료 = 전부 true) |
| scope (BE/FAB/MIX) | 그 회사가 **뒷공정에 얼마나 직결**되어 있는지 구분 (직결/공통 인프라/앞공정 병행) |
| popNow | **지금 몸값 ÷ 최신 영업이익** — 주가가 현재 싼지 비싼지 보는 배수 |
| 지금 종합 (qarpNow) | **지금 주가로 다시 계산한 종합점수·순위** — 원래 점수는 선정 당시 기준으로 따로 보존 |
| 시계열 | 공장 생긴 **이후에** 증설했다는 **시간 순서** 증거 |
